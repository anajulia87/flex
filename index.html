O que é display flex

Ela distribui o espaço extra entre os elementos flexíveis, permitindo centralizar, espalhar ou empacotar itens horizontalmente ou verticalmente.

O que é justify-content

Ele facilita o alinhamento, distribuição de espaço e criação de layouts responsivos, adaptando-se a diferentes tamanhos de tela sem necessidade de cálculos manuais complexos.

O que é aling-itens

Ela é usada para posicionar elementos verticalmente (quando em linha) ou horizontalmente (quando em coluna), controlando o alinhamento de todos os itens como um grupo.
